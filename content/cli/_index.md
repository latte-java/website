## Install

```bash
curl -fsSL https://lattejava.org/cli/install | bash
```

## Usage

```bash
# Create a new project
latte init

# Install a dependency
latte install org.apache.commons:commons-collections:3.1.0

# Build the project
latte build

# Run tests
latte test
```
