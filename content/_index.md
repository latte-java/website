---
title: "Latte Java"
description: "Latte aims to make Java simple and easy to use"
---

### 1. Install Java

```bash
curl -fsSL https://lattejava.org/javaenv/install | bash
```

```bash
javaenv install 25
```

```bash
javaenv global 25
```

### 2. Install Latte

```bash
curl -fsSL https://lattejava.org/cli/install | bash
```

### 3. Create a project

```bash
mkdir my-project && cd my-project
```

```bash
latte init
```
